## 감지 (Detect)

### 발견된 패턴
4개 스크립트에서 동일한 구조의 "━━━" 구분선 배너를 사용한다. 총 15회 출력. 구분선 길이(49자)와 스타일이 통일되어 있으나, 매번 raw echo로 출력한다.

### 발생 빈도
- 배너 구분선 출력: **15회** (install.sh 4회, x-verify.sh 3회, gap-check.sh 4회, init.sh 4회)

### 증거
```bash
# 4개 파일에서 동일한 패턴
```

## 제안 (Propose)

### 규칙 내용
현 시점에서는 **규칙화하지 않는다**.

이유:
1. 배너는 시각적 요소로, 함수화해도 가독성 향상이 크지 않다
2. 각 배너의 내용(제목)이 모두 다르므로 함수 인자가 복잡해진다
3. 001번 제안(common.sh)이 승인되면 그때 함께 포함하는 것이 적절하다
